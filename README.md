# MyBooks!

O MyBooks é uma plataforma onde você consegue buscar por livros e adicioná-los ao seus favoritos.



## Instalação

Use Yarn ou NPM para instalar os modules e inicializar o projeto.
```bash
yarn install
yarn start
```
ou
```bash
npm install
npm start
```
